The Project

The H0M3ST3AD Project is an on-going effort to make communication between you and your wifi-based projects effortless. Keeping all projects at a centralized place means you constantly have easy access to them. By keeping key details about your devices like the IP address and the communication protocols, the app can automatically fill in repetitive and uneccessary details so that you don't have to.

For now, the key features are keeping all your projects in a simple centralized location, diplaying this information in a condensed dashboard table, and providing a direct communication terminal that autofills repetitive information. In the future, we hope to support storing and executing custom functions, as well as data logging.

Important Note: Security is being constantly updated, but the application is has not reached a production release. As such, if you decide to use our service, please consider carefully the data that is transferred through this service.
